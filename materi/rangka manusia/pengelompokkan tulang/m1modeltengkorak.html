PETUNJUK PENGGUNAAN 3D MODEL
  • Tahan "klik kiri" pada Mouse/Trackpad untuk menggerakkan 3d model 360 derajat
  • Tahan "klik kanan" pada Mouse/Trackpad untuk mengubah posisi 3d model
  • Scroll "Track Ball" pada Mouse untuk zoom in / zoom out pada 3D model
  • Tombol "X" untuk kembali ke laman materi
